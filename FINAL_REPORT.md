## 🚀 الخطوات التالية

### 1️⃣ الإعداد المحلي (اختياري)
```bash
cd kira-bot
npm install
# عدّل config.json
# أضف appstate.json الخاص بك
node index.js
```

### 2️⃣ الرفع على GitHub
```bash
cd kira-bot
git init
git add .
git commit -m "🚀 Kira Bot - Ready to Deploy"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/kira-bot.git
git push -u origin main
```

### 3️⃣ الربط مع Render
1. اذهب إلى https://render.com/
2. New + → Web Service
3. اربط مستودع GitHub
4. اضبط الإعدادات:
   - Build: `npm install`
   - Start: `node index.js`
5. أضف APPSTATE في Environment Variables
6. Deploy!
